#  Number Guessing Game

A simple and fun Python game where the user tries to guess a randomly generated number within limited attempts.

---

##  Features

*  Random number generation (1–100)
*  Limited attempts (7 chances)
*  Hints (Too High / Too Low)
*  Replay option after game ends
*  Input validation (handles invalid inputs)

---

##  Technologies Used

* Python (Core concepts)
* Random module
* Loops & Conditional Statements

---

##  How to Run

1. Clone the repository:

```
git clone https://github.com/akhil78-bit/number-guessing-game.git
```

2. Navigate to the folder:

```
cd number-guessing-game
```

3. Run the game:

```
python game.py
```

---

## 🧠 How It Works

* The program generates a random number between 1 and 100.
* The user gets 7 attempts to guess the number.
* After each guess:

  * If the guess is low → “Too Low”
  * If the guess is high → “Too High”
* If guessed correctly → user wins 🎉
* If attempts finish → game over 😢
